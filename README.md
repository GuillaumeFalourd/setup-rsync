☞ Github actions to setup [Rsync](https://linux.die.net/man/1/rsync) (all os supported) 🔄

## 📚 Usage

### Ubuntu

```yaml
    runs-on: ubuntu-latest
    steps:
      - uses: GuillaumeFalourd/setup-rsync@v1
      - run: rsync --version
```

### MacOs

```yaml
    runs-on: macos-latest
    steps:
      - uses: GuillaumeFalourd/setup-rsync@v1
      - run: rsync --version
```

### Windows

```yaml
    runs-on: windows-latest
    steps:
      - uses: GuillaumeFalourd/setup-rsync@v1
      - run: rsync --version
```
